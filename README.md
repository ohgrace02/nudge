# Nudge — Interactive Prototype

Verified peer network for University of Georgia students.

## Local Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click "New Project" → Import this repo
4. Vercel auto-detects Vite — just click "Deploy"
5. Live URL in ~60 seconds

## What's in the Prototype

- Full onboarding with dynamic user data (name, major, year flow through every screen)
- Home screen with request feed and availability toggle
- Request help form (unified urgent/scheduled)
- Real-time matching simulation with live counter
- Match preview with trust signals
- Active session with live 15-min countdown timer and simulated chat responses
- Post-session feedback (rating, save connection, share contact)
- Explore / Campus Knowledge feed
- Messages inbox
- Profile / Settings

Built with React + Vite. Uses Nudge brand colors and SF Pro typography.
